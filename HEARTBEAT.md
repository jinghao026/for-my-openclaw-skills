# HEARTBEAT.md - 小秘的定期检查清单 💼

## 每日检查任务

### 记忆系统维护
- [ ] 检查 memory/YYYY-MM-DD.md 是否存在，不存在则创建
- [ ] 回顾今日重要交互，确保已记录到历史文件
- [ ] 如有重要事件，更新 MEMORY.md 长期记忆
- [ ] 检查是否有未完成的待办事项

### 代码与项目
- [ ] 检查 git 状态，如有未提交更改提醒 Boss
- [ ] 查看 skills/ 目录是否有新技能需要整理
- [ ] 检查 GitHub 仓库是否需要同步

### Boss关怀
- [ ] 检查是否有即将到来的日程/任务
- [ ] 超过8小时未互动时，主动问候

## 每周检查（周一执行）

### 记忆整理
- [ ] 回顾过去7天的 daily logs
- [ ] 提炼重要信息到 MEMORY.md
- [ ] 清理过期的临时信息
- [ ] 归档旧的记忆文件到 archives/

### 技能维护
- [ ] 检查已安装 skills 是否需要更新
- [ ] 查看是否有新的有用 skills
- [ ] 更新 skills/README.md 如有变化

### 系统健康
- [ ] 检查磁盘空间
- [ ] 验证备份完整性
- [ ] 审查安全日志（如有敏感信息泄露风险）

## 执行规则

**回复标准：**
- 如果无事需要关注 → 回复 `HEARTBEAT_OK`
- 如果有重要事项 → 汇报具体内容
- 如果发现问题 → 立即提醒 Boss

**时间窗口：**
- 深夜 (23:00-08:00) 除非紧急，否则静默
- 工作时间主动检查
- 尊重 Boss 的忙碌状态

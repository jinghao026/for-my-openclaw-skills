# AI 漫剧分镜生成器

将故事/剧本转换为专业漫画分镜脚本，包含镜头语言、画面描述和 AI 绘图提示词。

## 功能

- **故事解析**：自动拆解场景和镜头
- **分镜生成**：输出标准分镜表格（镜号、景别、角度、画面描述）
- **AI 提示词**：为每个镜头生成 Midjourney/Stable Diffusion 可用提示词
- **多种风格**：支持日漫、美漫、国漫等风格

## 使用方法

### 基础用法

输入故事梗概，生成分镜：

```
把下面这个故事变成漫剧分镜：

一个少年在雨夜发现一把发光的剑，当他握住剑柄时，
周围的时间突然静止，雨滴悬停在空中...
```

### 指定风格

```
生成国漫风格的分镜：

[你的故事内容]
```

### 指定镜头数量

```
把下面这个故事分成8个镜头：

[你的故事内容]
```

## 输出格式

每个分镜包含：
- **镜号**：第几镜
- **景别**：特写/近景/中景/全景/远景
- **角度**：平视/俯视/仰视/倾斜
- **画面描述**：场景、角色、动作
- **对白/旁白**：台词或 narration
- **AI 提示词**：可直接用于 Midjourney/SD

## 示例

输入：
```
女孩在咖啡馆偶遇前男友，两人尴尬地对视...
```

输出：
```

| 2 | 中景 | 平视 | 女孩愣住，目光锁定角落里的前男友 | - | anime girl surprised expression, looking at someone, cafe background, emotional moment --ar 16:9 |

```

## 支持的漫剧风格

- `日漫` — 日式动画风格，清新唯美
- `美漫` — 美式漫画，线条硬朗，对比强烈
- `国漫` — 中国漫画风格，古风/现代皆可
- `韩漫` — 韩式条漫，精致唯美
- `黑白` — 传统黑白漫画，强调线条和网点

## 提示词技巧

你可以添加以下关键词来细化输出：
- `增加细节` — 更详细的画面描述
- `强调表情` — 重点描述角色表情变化
- `动作镜头` — 增加动态感和速度线
- `氛围优先` — 强调场景氛围和光影
